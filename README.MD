# The Shop Project

### Project initialize instruction

create a file inside config with name common.php and set your db config like the
example below.

```php
<?php

return [
    'db' => [
        'server' => 'localhost',
        'user' => 'your_username',
        'password' => 'your_password',
        'database' => 'your_database',
    ]
];
```
